Submit New Results to the Leaderboard

We welcome results from new detectors or new backbone LLMs evaluated with the standard OpenHalDet protocol. To ensure comparability, please use the provided pipeline for response generation, annotation, and evaluation.

  1. Run the OpenHalDet pipeline: python datasets_v1/generate_pipeline.py
  2. Evaluate your detector: python baseline_detectors/runner.py
  3. Open a GitHub Issue with the title [Leaderboard] Method Name — Backbone
  4. Include: method name, access regime, AUROC per dataset, backbone, and a link to your paper or code
