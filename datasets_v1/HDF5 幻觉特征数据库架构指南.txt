===================================================================
📖 HDF5 隐藏层特征数据库 (Hidden States Database) 架构说明
===================================================================
本数据库采用层级目录结构 (Hierarchical Data Format) 存储大模型生成时的内部神经元激活状态。
所有张量数据均以无损极度压缩的二进制 `bfloat16` (|V2) 格式存储。

【目录拓扑结构树】
/ (Root 根目录)
│
├── [Global Attributes] 🌍 全局属性 (记录实验的“基因条码”)
│   ├── llm_model: "meta-llama/Llama-3.2-1B-Instruct"  (使用的待测模型)
│   ├── dataset_source: "01_unified_prompts.jsonl"     (数据源文件)
│   ├── layer_config: '{"mode": "middle", "count": 3}' (提取了哪几层)
│   └── token_config: '{"mode": "backward", "count": 5}'(提取了哪些Token)
│
├── xsum_validation_000000/ (📂 样本级目录：以 Sample ID 命名)
│   │
│   ├── [Attributes] 📝 样本级属性
│   │   ├── original_prompt: "Summarize this: The UN reported..." (输入提示词)
│   │   └── model_output: "The UN warned about climate change." (模型完整回答)
│   │
│   └── generated_tokens/ (📂 Token 容器组)
│       │
│       ├── token_012/ (📁 具体的 Token 文件夹)
│       │   ├── [Attributes] 🏷️ Token 属性
│       │   │   ├── text: " climate"       (具体的文本片段)
│       │   │   ├── token_id: 8523         (模型词表中的 ID)
│       │   │   ├── forward_idx: 12        (这是模型生成的第12个词)
│       │   │   └── backward_idx: -4       (这是倒数第4个词)
│       │   │
│       │   ├── layer_07 (🧮 数据集 Dataset：第7层特征，形状 (2048,))
│       │   ├── layer_08 (🧮 数据集 Dataset：第8层特征，形状 (2048,))
│       │   └── layer_09 (🧮 数据集 Dataset：第9层特征，形状 (2048,))
│       │
│       └── token_013/ (📁 下一个 Token 文件夹)
│           └── ... (结构同上)
│
├── xsum_validation_000001/ (📂 第二个样本目录)
└── ... (更多样本)
